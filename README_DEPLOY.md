# GitHub Pages deployment
1. Create a GitHub repository, for example `packagingfactorydirect`.
2. Upload all files in this folder to the repository root.
3. Go to Settings > Pages > Deploy from branch > main > root.
4. Add custom domain: packagingfactorydirect.com.
5. In your domain DNS, point to GitHub Pages according to GitHub's current instructions.
